# Version 5.0

## camelCase

Use camelCase throughout the repository, wherever possible.

Before:

```ts
new Voice({
  num_beats: 4,
  beat_value: 4,
});
```

After:

```ts
new Voice({
  numBeats: 4,
  beatValue: 4,
});
```

## `stavenote.ts`

- Remove getter `StaveNote.note_heads: NoteHead[]`. It was previously deprecated. Use `StaveNote.noteHeads` instead.
